    About

    This plugin/channel creates a quick way to access the stream at nickreboot.com natively in Plex.

    Usage

    Tested on Plex Media Server 0.9.9.14 on Windows 7. Requires Transcoding to be ON.
    Clients Tested included Plex Media Center, RasPlex, Roku, and iOS

    Install

    To install, rename the encasing top level folder "nickreboot.bundle", then place it it one of the following folders, then restart Plex Media Server.

    • Windows vista/7/8/8.1 or later: Copy nickreboot.bundle to C:\Users*your username*\AppData\Local\Plex Media Server\Plug-ins

    • Windows XP: Copy nickreboot.bundle to C:\Documents and Settings*your username*\Local Settings\Application Data\Plex Media Server\Plug-ins

    • Mac/Linux: Copy nickreboot.bundle to ~/Library/Application Support/Plex Media Server/Plug-ins
